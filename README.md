### FCC URL Shortener Microservice

Pass a URL as a parameter and receive a shortened URL in JSON response.

Passing an invalid URL will return a JSON response with an error.

Visiting the shortened URL, will redirect to original link.

#### Start Project

```
git clone https://github.com/Concepsheon/fcc-url-shortener.git
```

Move to folder and run ```npm install``` and ```node app``` to run the project
